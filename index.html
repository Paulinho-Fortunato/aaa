QRGen - Gerador e Leitor de QR Codes

QRGen é uma ferramenta online gratuita e de código aberto que permite gerar e ler códigos QR de forma simples e segura. Com uma interface intuitiva e moderna, você pode criar códigos QR personalizados para URLs, contatos, redes sociais, redes Wi-Fi e muito mais.

Funcionalidades
  • Gerar QR Codes para URLs, textos, contatos (vCard), redes sociais e configurações Wi-Fi
  • Leitor de QR Codes via câmera ou upload de imagens
  • Personalização de cores, tamanho e margens do QR Code
  • Download do QR Code gerado como imagem PNG
  • Interface multilíngue (português e inglês)
  • Modo claro e escuro para melhor experiência do usuário
  • Totalmente responsivo e acessível
Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias web modernas, incluindo HTML5, CSS3, JavaScript, Bootstrap para estilização responsiva, qrcode.js para geração de códigos QR e jsQR para leitura de códigos QR.

Privacidade e Segurança

A privacidade dos usuários é nossa principal preocupação. Todos os dados são processados localmente no seu dispositivo, sem envio para servidores externos. Nenhum dado é armazenado ou rastreado.
